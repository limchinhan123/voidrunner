# Touch the Stars

Generative art arcade experience built with HTML5 Canvas and JavaScript.

- Tap or click anywhere to plant glowing flowers
- Drag to paint trails and sparkles
- Responsive on desktop and mobile
- Audio built with the Web Audio API
- Single-file browser game; no dependencies

## Play

Live: https://limchinhan123.github.io/voidrunner/

## Tech

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API
- GitHub Pages

## Controls

- Click or tap to plant a flower
- Click or drag to leave a sparkle trail

## Notes

This project is part of my portfolio. The code focuses on particle systems, animation loops, and lightweight client-side interaction design.
